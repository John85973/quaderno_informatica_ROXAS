SPIEGAZIONE

Il codice crea una pagina web per registrare automobili e proprietari in un database MySQL.

Inizia con un modulo HTML dove l'utente inserisce i dati dell'automobile, come targa, modello, marca e colore.

Contemporaneamente, l'utente puo inserire i dati del proprietario, come codice fiscale, nome e cognome.

Quando il modulo viene inviato, i dati vengono inviati al server tramite il metodo POST.

Un file PHP riceve questi dati e si connette al database MySQL usando le credenziali fornite.

Il PHP esegue due query per inserire i dati nelle rispettive tabelle: una per le automobili e una per i proprietari.

Una volta completato l'inserimento, viene mostrato un messaggio di conferma dell'avvenuta registrazione.

Infine, la pagina consente di visualizzare le tabelle con i dati gia registrati.


Per far funzionare il codice:

1. Apri il tuo browser e accedi a `http://localhost/phpmyadmin/`.

2. Clicca su "Nuovo" nella barra laterale sinistra per creare un nuovo database.

3. Dai un nome al database come in questo caso "AutomobileProprietario" e clicca su Crea.

5. Crea le tabelle Automobile, Proprietario e Possesso con gli attributi seguendo il problema

6. Clicca esegui per creare le tabelle

PROBLEMA SCELTO:

Si vuole progettare una base di dati per gestire le informazioni di automobili, descritte da una targa, un

modello, una marca e un colore e dei rispettivi proprietari, che possono anche essere piu di uno, identificati da

un codice fiscale, nome e cognome, sia come proprietari attuali che come eventuali proprietari passati, ne

qual caso e anche necessario indicare il periodo di possesso dell'autovettura. Ricordiamo che un proprietari in

puo possedere piu automobili.
